Інтелектуальний асистент для жителів Києва. Моніторинг безпеки, новин, інфраструктури та дозвілля в режимі реального часу.

Команди Відкрити Бота

# МОДУЛІ СИСТЕМИ

Park Finder

Інтелектуальний пошук парків та зон відпочинку. Аналізує геолокацію, відгуки та фотографії для підбору найкращих місць.

Gastro Guide

Рандомайзер закладів харчування з фільтрацією по рейтингу. Надає актуальні фото, ціни та графік роботи.

News Core

Агрегатор новин з 10+ перевірених джерел. AI-фільтрація спаму та дублікатів для надання лише важливого контенту.

Energy Monitor

Парсинг графіків відключень ДТЕК та Yasno. Автоматичне розпізнавання тексту з зображень графіків.

# СПИСОК КОМАНД

/start System
Ініціалізація бота. Реєструє користувача в базі даних usersinfo.json та відкриває головне навігаційне меню.
📰 КиНовини News
Запускає алгоритм збору новин:
  1. Підключається до Telegram API через Telethon.
  2. Сканує останні повідомлення з 10 каналів Києва.
  3. Фільтрує рекламу та спам за ключовими словами.
  4. Обирає 3 випадкові важливі новини, яких ви ще не бачили.
🔔 Тривога Security
Звертається до Siren API в реальному часі.
  • Показує статус повітряної тривоги у м. Київ.
  • Відображає тип загрози (Ракети/БПЛА/Артилерія).
  • Надає кнопку з посиланням на інтерактивну карту.
🌧️ Погода в Києві Utility
Запит до WeatherAPI для отримання метеоданих:
  • Температура (фактична та "відчувається як").
  • Вологість, швидкість вітру та опис погодних умов.
  • Автоматичний підбір емодзі під поточний стан погоди.
🚨 Графіки Відключень Utility
Моніторинг офіційного каналу ДТЕК:
  • Знаходить останнє повідомлення з графіком.
  • Завантажує актуальне фото розкладу.
  • Очищає текст від маркетингового сміття, залишаючи лише суть.
🍽 Рандомний ресторан Leisure
Обирає випадковий заклад з локальної бази restaurants.json:
  • Генерує картку закладу (Назва, Рейтинг, Ціни).
  • Підтягує до 10 фотографій інтер'єру/їжі.
  • Показує точну адресу та категорії кухні.
🌳 Рандомний парк Leisure
Пошук зелених зон з бази parks.json:
  • Фільтрація "фейкових" парків (парковки, магазини).
  • Відображення статусу (Відчинено/Зачинено).
  • Надсилання фото місцевості для оцінки атмосфери.

# ЯК КОРИСТУВАТИСЬ

01. Знайдіть бота в Telegram

Відкрийте Telegram і введіть у пошук юзернейм бота або перейдіть за прямим посиланням:

@my_kyiv_bot
Перейти до бота

02. Запустіть бота

Натисніть кнопку START внизу екрану. Бот відкриє головне меню з доступними функціями.

03. Оберіть потрібну функцію

Використовуйте кнопки меню для отримання інформації. Наприклад, натисніть "📰 КиНовини", щоб отримати дайджест подій.
